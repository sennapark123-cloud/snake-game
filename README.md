# Snake Game 2026

TypeScript + HTML5 Canvas로 제작한 모던 스네이크 게임

---

## 실행 방법

```bash
# 의존성 설치 (최초 1회)
npm install

# 개발 서버 실행
npm run dev
```

브라우저에서 **http://localhost:5173** 접속

---

## 조작키

| 키 | 동작 |
|----|------|
| `W` / `↑` | 위 이동 |
| `S` / `↓` | 아래 이동 |
| `A` / `←` | 왼쪽 이동 |
| `D` / `→` | 오른쪽 이동 |
| `Space` / `Enter` | 시작 / 재시작 |

## 기능

### 레벨 시스템
점수에 따라 레벨이 올라가며 속도가 증가합니다.

| 레벨 | 필요 점수 |
|------|-----------|
| 1 | 0 |
| 2 | 100 |
| 3 | 250 |
| 4 | 500 |
| 5 | 800 |
| 6 | 1,200 |

### 파워업 아이템
먹이 획득 시 25% 확률로 보드에 등장합니다.

| 아이콘 | 이름 | 효과 |
|--------|------|------|
| ❄ | SLOW | 이동 속도 감소 |
| ★ | INVINCIBLE | 벽 통과 + 무적 |
| ×2 | DOUBLE SCORE | 점수 2배 |
| ↓ | SHRINK | 뱀 길이 5칸 감소 |

### 리더보드
게임 오버 후 최고 기록 달성 시 이름을 입력하여 상위 10위까지 저장됩니다. (localStorage)

### 애니메이션 효과
- 먹이·파워업 획득 시 파티클 이펙트
- 레벨 업 플래시 애니메이션
- 무적 상태 무지개 색상
- 파워업 소멸 전 점멸 경고

## 기술 스택

- **언어**: TypeScript 5.x
- **렌더링**: HTML5 Canvas API
- **빌드 도구**: Vite 5.x
