## 3. 게임 스펙

### 3-1. 캔버스

| 항목 | 값 |
|------|----|
| 해상도 | 600 × 500 px |
| 그리드 | 30열 × 25행 |
| 셀 크기 | 20 × 20 px |
| 배경색 | `#1a0514` (다크 핑크) |

### 3-2. 조작

| 키 | 동작 |
|----|------|
| `W` / `↑` | 위 이동 |
| `S` / `↓` | 아래 이동 |
| `A` / `←` | 왼쪽 이동 |
| `D` / `→` | 오른쪽 이동 |
| `Space` / `Enter` | 시작 / 재시작 / 확인 |

### 3-3. 레벨 시스템

| 레벨 | 필요 점수 | 이동 속도 (ms/tick) |
|------|-----------|---------------------|
| 1    | 0         | 150                 |
| 2    | 100       | 130                 |
| 3    | 250       | 110                 |
| 4    | 500       | 90                  |
| 5    | 800       | 75                  |
| 6    | 1,200     | 60                  |

레벨 업 시 1.5초간 레벨 업 애니메이션 표시 후 게임 재개.

### 3-4. 점수

| 이벤트 | 점수 |
|--------|------|
| 먹이 획득 (기본) | +10 |
| 먹이 획득 (×2 효과 중) | +20 |

---

## 4. 파워업 아이템

먹이 획득 시 **25% 확률**로 보드에 스폰. 최대 2개 동시 존재. 30 tick 후 자동 소멸(소멸 10 tick 전 점멸).

| 아이콘 | 타입 | 색상 | 효과 | 지속 |
|--------|------|------|------|------|
| ❄ | SLOW | `#4fc3f7` | 이동 속도 1.7배 감소 | 20 tick |
| ★ | INVINCIBLE | `#ffd54f` | 벽 통과 (반대쪽으로 워프), 자기 몸 충돌 무시, 무지개 색상 | 18 tick |
| ×2 | DOUBLE_SCORE | `#ce93d8` | 먹이 점수 2배 | 25 tick |
| ↓ | SHRINK | `#80cbc4` | 뱀 길이 5칸 즉시 감소 | 즉시 |

## 5. 게임 상태 흐름

```
WAITING
   │  Space / Enter
   ▼
PLAYING ──── 레벨 달성 ──▶ LEVEL_UP ──▶ PLAYING
   │
   │ 벽·자기 몸 충돌
   ▼
GAME_OVER
   │  Space / Enter
   ├── 최고 기록이면 ──▶ 이름 입력 (askingName)
   │                           │ Enter
   │                           ▼
   └─────────────────▶ LEADERBOARD
                           │  Space / Enter
                           ▼
                        PLAYING (재시작)
```

---

## 6. 파티클 시스템

- 먹이 획득: 빨간 파티클 14개 방사
- 파워업 획득: 파워업 색상 파티클 18개 방사
- 중력 적용 (vy += 0.12/frame), 마찰 (vx *= 0.95)
- 수명: 약 28프레임 (life 1.0 → 0, 매 프레임 -0.035)

---

## 7. 리더보드

- 저장소: `localStorage` (`snake2026_scores`)
- 최대 저장: 상위 10위
- 저장 항목: 이름(최대 10자), 점수, 레벨, 날짜
- 신기록 조건: 0점 초과 AND (10위 미만 OR 10위 점수 초과)
